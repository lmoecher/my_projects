## Formatting Description

* *Formatting* elements were designed to have the *HTML* document display special text.

## Different Formatting Elements

* "*b*" - Bold text
* "*strong*" - Important text
* "*i*" - Italic text
* "*em*" - Emphasized text
* "*mark*" - Marked text
* "*small*" - Smaller text
* "*del*" - Deleted text
* "*ins*" - Inserted text
* "*sub*" - Subscript text
* "*sup*" - Superscript text
